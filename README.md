# μChordbot

Microtonal chord progression editor and harmonic prototyping PWA.

μChordbot is a browser-based tool for:

- microtonal chord design
- harmonic progression prototyping
- relative progression presets
- chord progression editing
- xenharmonic experimentation
- WebAudio-based realtime playback

Supports:
- arbitrary microtonal roots
- relative progression insertion
- progression presets
- microStep-based pitch system
- chord libraries
- progression transposition
