# muChordbot Behavior Spec

## 数直線
- cent 入力は小数を保持する
- スナップ変更時は pitchDraft を再計算する
- オクターブ別の activeNotes マーカーは縦方向にずらして重なりを見分けられるようにする

## activeNotes
- リロード時に activeNotes は復元しない
- 行左の `×` で削除する
- `一括削除` で全削除する

## コード呼び出し
- ルート音高入力が優先
- 直接入力が空のときだけルートプリセットを参照
- どちらも無い場合は案内文を出し、展開ボタンを無効化する

## コード進行
- 選択セルがある場合、編集内容はそのセルへ即時反映する
- 再生中に並び替え / 削除が起きたら安全側で停止する
- 並び替え後も `selectedPartId` は同一セルを指し続ける
