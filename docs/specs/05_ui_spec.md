# muChordbot UI Spec

## 共通
- 上部に `元に戻す / やり直す / Master Vol.` を常設する
- タブは `音高 / コード進行 / 設定 / 入出力`
- 表示文言は日本語基準で統一する

## 音高タブ
- `オクターブ / スナップ cent / cent / microStep`
- `持続音` チェックボックス、波形、activeNotes 音量
- 数直線右上に `cent / microStep` readout を重ね表示
- activeNotes 行は `× / 音高表示 / 音高プリセット入力群 / 追加`

## コード保存 / 呼び出し
- コード保存は `コード ID / コード名 / 基準音 / タグ / 各音ラベル / メモ`
- コード呼び出しは `保存済みコード / ルート音高 / ルートプリセット`
- 呼び出し前に案内文と状態文を表示する

## コード進行
- 上部に `前へ / 次へ / 再生 / 停止 / ループ / 列数`
- セル一覧は Chordbot 風に簡潔表示する
- 下部エディタで root 直接入力、b/♮/#、オクターブ、コード選択、拍選択を行う
- モバイルではメニューと編集パネルを折りたたみ可能にする
