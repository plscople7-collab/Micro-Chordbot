# muChordbot Requirements

## 必須要件
- 微分音を `1oct = 3600 microStep` で扱えること
- 数直線上で音高を編集し、その場で発音確認できること
- activeNotes を保持し、個別削除と一括削除ができること
- activeNotes から音高プリセットとコードを保存できること
- 保存済みコードを別ルートへ呼び出して activeNotes に展開できること
- コード進行をセル単位で編集、再生、並び替えできること
- Undo / Redo で主要操作を巻き戻せること
- `.mcb` / `.mcbl` の import / export ができること

## UI 要件
- 主要表示は日本語で統一すること
- スマホ / タブレット / PC で同じ情報へ到達できること
- モバイルでは折りたたみやメニュー化で情報量を維持すること
- マスター音量は全タブ共通表示とすること

## 配布要件
- localhost 起動を正式サポートすること
- 同一 Wi-Fi の LAN URL で実機ブラウザ確認できること
- HTTPS 公開時に PWA install が可能であること
