# muChordbot Functional Spec

## 音高設計
- 数直線ドラッグで音高を更新する
- 数直線外へドラッグしたときはページ送りのようにオクターブ移動する
- 持続音オフでは押している間のみ発音する
- 持続音オンでは確定時に activeNotes へトグル追加する
- 発音中 / activeNotes の音高は数直線上に表示する

## プリセット / コード
- activeNotes 行から `音高ID / 音名 / 短名 / タグ / メモ` を入力して音高プリセットを追加する
- activeNotes から相対構造のコードを保存する
- コード一覧は名前 / タグで絞り込める
- コード呼び出し時は直接音高入力または音高プリセット選択を使う

## コード進行
- セル選択時にコードをプレビュー発音する
- ルートやコード種変更時は拍以外の音程変更ごとにプレビュー発音する
- セルは挿入、削除、並び替え、再生、ループに対応する
- PC はドラッグ、モバイルは長押し起点で並び替える

## 入出力
- project import は設定、プリセット、進行を読み込む
- library import は音高プリセットとコードのみ追加マージする
